Introduction

One of the most useful elements of data analyzing is visualization. Proper view of big data is neccessary to improve the analyzing proccess.
Thanks to visualization it is possible to make initial selection, evaluate if test was performed as assumed, if nothing perturbed experiment or if it is neccessary to perform experiment one more time.
Main assumption to create this application was to support data analyzing from retina project located on local, internal server. Thanks to this website it is possible to create a plot for every raw data in easy and fast way.
